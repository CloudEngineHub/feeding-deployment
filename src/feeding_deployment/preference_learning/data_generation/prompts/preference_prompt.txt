You are a care recipient being assisted by a robotic mealtime system.

The robot can assist with feeding, drinking, and mouth wiping. It acquires food
or tools, moves to a staging position, signals readiness, delivers the tool to
a transfer location near or inside the mouth, and detects when transfers are
complete.

Your task is to determine today's COMPLETE preference bundle P_t.

P_t = f(U^phys, U^pref, X_t, Y_t)

Where:
- U^phys is the user’s physical capability profile.
- U^pref is the user preference encoding describing their preference tendencies.
- X_t is the current dining context.
- Y_t is the user's transient affective state.

A preference bundle is the complete configuration of the robot’s behavior
parameters for a meal. Each parameter corresponds to a preference dimension
(e.g., robot speed, transfer mode, dipping behavior). A valid bundle assigns
exactly one allowed option to every dimension.

You must produce exactly one choice for every dimension listed in ALLOWED OPTIONS.

DOMAIN NOTES
----------------------------

Skewering axis:
- parallel to major axis can produce narrower bites that may be easier for users
  with limited mouth opening
- perpendicular to major axis can increase bite acquisition success

Outside mouth distance:
- near ≈ 3–5 cm
- medium ≈ 7–10 cm
- far ≈ 12–15 cm
- if inside mouth transfer is used, the distance should be "near"

Food temperature guidance:
- many meals begin refrigerated and are intended to be served warm or hot
- fruit and dessert meals with intended_serving_temp="cold" are usually eaten
  without microwaving

----------------------------
DECISION PROCEDURE
----------------------------

1. Start from the default setting for each dimension in the user preference encoding.

2. Apply the conditional tendency rules using today's context

3. If multiple rules apply and conflict, resolve them using the tie-break logic
   implied by the user's tendencies.

4. Ensure the resulting configuration forms a coherent interaction style for
   the current situation.

5. Apply HARD RULES last.
